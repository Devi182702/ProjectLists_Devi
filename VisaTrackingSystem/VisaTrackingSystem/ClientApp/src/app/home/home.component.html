Welcome to Visa Tracking System!

Welcome to your new single-page visa application!

  • End User: Human Resources Department
  • Purpose: To keep track of the employees visa related details
  • Technologies Used:
    • ASP.Net Core
    • EF core
    • Angularjs 5
    • HTML
    • TypeScript

To help you get started, we've also set up:

  • Client-side navigation. For example, click Visa-Detail on the left side menu to view all the visa records.

The Visa Tracking System is designed extensively for HR personnel for handling visa information about their employees.

  1. View all the Visa Records
  2. Create a new Visa Record for an employee
  3. Edit a existing Visa Record with some new values
  4. Delete an Visa Record
